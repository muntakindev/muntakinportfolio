# Muntakin Futuristic Portfolio

A premium animated portfolio website built with HTML, CSS, JavaScript, and a Node.js backend for contact submissions.

## Features

- Futuristic dark UI with glassmorphism and glowing accents
- Full-screen hero, animated typing, and motion particles
- About, skills, projects, services, testimonials, and contact sections
- Project details modal with live demo and project links
- Floating cursor, smooth reveals, and scroll-triggered animation
- Responsive mobile/tablet/desktop layout
- Contact form powered by Node.js endpoint

## Run Locally

1. Install dependencies:
   `npm install`
2. Start the server:
   `npm start`
3. Open `http://localhost:3000`

## Optional helper modules

- `java/PortfolioInfo.java` — example Java data model for portfolio metadata
- `cpp/contact_validator.cpp` — example C++ utility for validating contact input
